# PoolProxy
 稳定的矿池转发程序，纯中转模式无任何抽水；支持开机自动启动；支持ETH、ETC、ERGO、LTC、RVN等主流币种；支持矿池加密转发；支持SSL、自有证书；
 
[Telegram讨论组](https://t.me/PoolProxy)

### 注:
* 加密模式不需要VPS服务器,需要在本地局域网运行本软件
* 本软件仅为转发程序,需配合轻松矿工、开源旷工等挖矿软件使用
* 本软件为纯中转模式无任何抽水,没有软件开发费
